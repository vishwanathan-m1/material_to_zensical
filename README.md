# Zensical Docs

### About this Repository

- This is a project repository created to generate modern document sites made with Zensical on top of Material for MkDocs. 

- The code files are hosted on GitHub (open-source) public server.

- _GitHub Pages_ is designed to host the project pages from GitHub repository.

- _Build and deployment_: Site is currently being built from the _root_ folder in the _gh-pages_ branch.

- GitHub repository link: [https://github.com/vishwanathan-m1/material_to_zensical](https://github.com/vishwanathan-m1/material_to_zensical)

- Latest live document site link: [https://vishwanathan-m1.github.io/material_to_zensical](https://vishwanathan-m1.github.io/material_to_zensical)
